üìÅ Project Overview

This project simulates a 4-DoF articulated manipulator using Python, NumPy, and Matplotlib.
The robot configuration follows the joint order:

Revolute

Revolute

Revolute

Prismatic

This is sometimes called an RRRP manipulator.
It provides horizontal reach through the revolute joints and vertical motion through the prismatic axis, allowing the arm to pick objects from the floor and place them onto multi-level shelves.

ü¶æ Robot Model & Kinematics

The robot class (ArticulatedRRRP) implements:

Forward Kinematics (FK):
Used to compute and visualize all joint positions.

Inverse Kinematics (IK):
The IK solves the 2-link planar geometry for x-y positioning,
and sets the prismatic joint extension to reach the required z-level.

The IK also includes reach clamping ‚Äî whenever a target is outside the workspace, the solver automatically adjusts the position so the manipulator moves as close as physically possible.
